Class MitivWiener

  • java.lang.Object
    • EzPlug
      • plugins.mitiv.deconv.MitivWiener

  • public class MitivWiener
extends EzPlug
    MiTivWiener is a deconvolution tool based on linear methods. It's goal is to offer a simple interface for the user with a minimal amount of parameters to change. It includes 3 methods with two of them that are nearly equals: wiener, quad. CG is the linear conjugate gradients, for the on linear part see the other plugin: total variation.
